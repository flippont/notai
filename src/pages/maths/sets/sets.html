A set is a collection of things. The “things” that make up a set is called an element. For example, if you had a fourth year hockey team with Paul, Peter, and Paula, they would be elements in the set for that fourth year hockey team. If you were to call this set A, set A would be written as:

A = {Paula, Paul, Peter}

Every element we treat in a certain question is called a universal set, which is written with the letter U.

Other Examples

If you were to have a set of all prime numbers under 20, which you call C, you can write that as:

C = {2, 3, 5, 7, 11, 13, 17, 19}

Now, if this was a set of all primes without limits, it would have an endless set of elements. This is called an infinite set.

Key Terms

There are a couple of key terms that are associated with sets. Here is a list down below.

Term

Symbol

Definition

Empty set

{} / φ

A set without any elements

Membership

x ∈ A

x is an element of A

Cardinality

|A|

The number of elements in set A

Intersection

A ∩ B

A set of all elements that are in both A and B

Union

A ∪ B

The set of all elements that are in A or B or both

Subsets

A ⊆ B

All elements in set A are included in set B
