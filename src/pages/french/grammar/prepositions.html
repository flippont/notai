De : of / some 

Changes depending on number of noun 
Je mange de… 

De+ le => du (E.g. Je mange du pain) 
De + la => de la (E.g. Je mange de la banana) 
De + les  => des (E.g. Je mange des pommes) 
De + l’ => De l’ (E.g. Je bois de l’eau) 

A :  

Aller a = to go to 
Etre a  = To be at 
Jouer a = Play + Sport 
Habiter a = Live at 

A + Le => Au (E.g. Je vais au cinema) 
A + La => A la (E.g. Je vais a la piscine) 
A + Les = Aux (E.g. Je vais aux mangeons) 

Quel : Which  

Quel => Masc Singular 
Quelle => Fem Singular 
Quels => Masc Plural 
Quelles => Fem Plural 

Tout : Every/All 

Tout => Masc singular (E.g. Tout le mande) 
Toute => Fem singular (E.g. Toute la classe) 
Touts => Masc plural (E.g. Tous le jours) 
Toutes => Fem Plural (E.g. Toutes les filles)
